I used to plan my day in obsidian or MsWord by creating tables and writing every single thing by myself and organizing things myself
but I realized that it takes a lot of time and if I have to change anything it becomes complex and i have to start from scratch,
so i decided to create a web for this use and I created this website

Features:

    Add tasks with a name, start/end time block, category tag, note, and color
    Auto-calculates and displays duration of each time block
    Mark tasks as done (with strikethrough)
    Edit any task at any time
    Delete tasks with a confirmation prompt
    Drag and drop to reorder tasks
    Filter tasks by All / Pending / Done
    Sort by custom order, start time, or duration
    Stats bar showing total, done, planned hours, and remaining tasks
    Progress bar with percentage
    Tasks persist via localStorage — survive reloads and browser restarts
    "Saved" confirmation flashes after every change
    Works fully offline, no internet needed after download

Limitations:

    Single device only — no sync across phone, tablet, or other computers
    No date separation — all tasks live in one list regardless of day
    No recurring tasks — repeated daily tasks must be re-added manually
    No reminders or notifications
    Tied to one browser — tasks don't carry over if you open the file in a different browser (e.g. Chrome vs Firefox)
    If you clear browser site data/cache, all tasks are permanently lost with no recovery
    No export or backup option — no way to save tasks to a file as a safety net
    No subtasks — each task is a flat single item

It works perfects for me and I would keep using it and if you want you can use it too :)
